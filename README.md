Projet de Collecte des Offres d'Emploi en Mauritanie
Ce projet vise à agréger les offres d'emploi disponibles en Mauritanie à partir de diverses sources en ligne. L'objectif est de faciliter la recherche d'emploi pour les utilisateurs en centralisant les offres sur une plateforme unique.

Technologies Utilisées
Frontend : Réalisé avec React, offrant une interface utilisateur interactive et dynamique.
Backend : Développé avec Spring, assurant une gestion robuste et sécurisée des données.
Base de Données et Stockage : Utilisation d'Alfresco pour le stockage des données, garantissant une gestion efficace des documents et des métadonnées.
Fonctionnalités Clés
Scraping Automatique : Extraction automatique des offres d'emploi à partir de divers sites web, assurant une mise à jour continue et exhaustive des annonces.
Recherche et Filtrage : Les utilisateurs peuvent rechercher et filtrer les offres d'emploi par critères spécifiques (localisation, type de poste, secteur d'activité, etc.).
Notifications : Les utilisateurs peuvent s'abonner pour recevoir des notifications lorsque de nouvelles offres correspondant à leurs critères sont publiées.
